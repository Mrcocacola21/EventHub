# Security Policy

## Current Security Measures

- Env-based secrets.
- JWT authentication.
- Role-based access.
- Object-level permissions.
- Atomic booking.
- Overselling protection.
- QR repeated-use protection.
- Audit logs.
- Request id.
- Protected media endpoints for PDF download.
- `.env` ignored.

## Reporting Security Issues

This is a portfolio project. For real deployments, configure responsible disclosure contact.
